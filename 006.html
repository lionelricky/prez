PHASE 2

Intranet Application

Create the intranet portion for employees and administrators. (intra.dtdesign.com or www.dtdesign.com/admin). Establish roles.

  • Employee directory.
  • Company announcements/events.
  • Administrative applications (e.g. create contracts, quotes, invoices, add Employees, manage roles).

Technologies used:

  • HTML, CSS, and VUE.Js.
  • Component frameworks such as Vuetify.js and Bootstrap.
  • PHP (Authentication and API)
  • MySQL database (Employee table, invoices table, client table, contracts table, etc.).
